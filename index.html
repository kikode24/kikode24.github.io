CosmoZone

Used Tools/Languages: JavaScript, HTML, PHP, Figma, BootStrap, PhotoShop

CosmoZone is a program that allows you to manage, add and edit events as well as interaction from a comments tab. Developed to help "Centros de Ciência Viva" (Live Science Centers) in Portugal organize events and preferences of its customers, it includes, in addition to the aforementioned, an area of administration with access to statistical data regarding the preference of events and user location.

PokéBase

Used Tools/Languages: JavaScript - React, HTML, Figma, BootStrap, CSS, PhotoShop

PokéBase is a project carried out in a more casual and personal context with the aim to simulate decks from the Pokémon TCG game. Incorporated with a login system by Firebase, it allows each account to have its own deck and compare it to the rest. It makes use of specific APIs related to Pokémon (PokéTCG and a Pokémon DataBase) to search for the Pokémons and relate them to their respective TCG cards.
